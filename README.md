# stayalive
A web app to keep a device awake during long running field work

# Installation
The app can be installed on the device as a progressive web app, aka "Add to Home".

# Usage
When active the stayalive app with keep your device awake and prevent the screen from locking.

  [!CAUTION]
  There are inherent security risks to this, so use the necessary precautions to secure your device.**

A maximum stay-awake time can be entered. It defaults to 30 minutes. Setting max time to 0 will
cause the device to remain on.

Toggle the Enable/Disable button to activate the stay awake timer. To disable the timer either toggle
the button, hide or close the browser tab.

Mobile device browsers do not allow web apps, so the stayalive app is designed
to run in split screen. Click the button to toggle compact mode and enable split screen. The 
process for entering split screen is different depending on your device.
